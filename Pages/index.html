Perguntas Frequentes

Como faço para usar o serviço?

Para começar a utilizar nosso serviço, siga estas simples etapas:

  1. 1. Crie uma conta em DINAMI$A.
  2. 2. Faça login na sua conta.
  3. 3. Conecte seus cartões e contas financeiras.
  4. 4. Explore as funcionalidades disponíveis para gerenciar suas finanças.

Quais são as taxas associadas?

Nossas taxas são transparentes e acessíveis. Oferecemos um modelo de precificação justo para garantir a melhor experiência para nossos usuários. Para detalhes específicos sobre taxas, consulte nossa seção de tarifas ou entre em contato conosco.

Como faço para recuperar minha senha?

Recuperar sua senha é fácil. Siga estas instruções:

  1. 1. Vá para a página de login.
  2. 2. Clique em "Esqueceu sua senha?".
  3. 3. Insira seu e-mail cadastrado.
  4. 4. Siga as instruções enviadas por e-mail para redefinir sua senha.
